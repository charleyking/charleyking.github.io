ubuntu下软件安装方法

Mar 14, 2016 | Hits

ubuntu下安装软件有三种方式，分别为在线安装apt-get方式，软件商店安装方式和

1. 软件商店安装方式

  • 这种方式对经常使用windows系统的同学来说最为简单，因为其界面是图形化的。现在的ubuntu系统默认就有软件商店，所以只要在里面搜索一下需要的软件，点击安装就可以。唯一的缺点就是很多不太知名或者刚出来的软件搜索不到，在这种情况下。我们只能从这些软件的官网上下载对应的安装文件或者压缩包，然后自己手动安装了。

2. 在线安装apt-get方式

  • 这种方法是最方便的了，因为只需要敲一行命令就好了。

  • sudo apt-get install packagename

  • 关于apt-get的其他方法，该命令一般需要root权限，所以前面要添加sudo命令

  1. apt-cache search package 搜索包
  2. apt-cache show package 获取包的相关信息
  3. apt-get install package 安装包
  4. apt-get -f install 修复安装
  5. apt-get remove package 删除包
  6. apt-get update 更新源
  7. apt-get upgrade 更新已安装的包
  8. apt-get dist-upgrade 升级系统
  9. apt-cache depends package 了解使用依赖
  10. apt-cache rdepends package 是查看该包被哪些包依赖
  11. apt-get source package 下载该包的源代码
  12. apt-get clean && sudo apt-get autoclean 清理无用的包
  13. apt-get check 检查是否有损坏的依赖

3. 手动安装方式

  • “deb格式文件”：这种文件格式和windows下的.exe格式是一样的，是ubuntu系统下的可执行安装文件，直接双击即可执行安装程序。另外也可以使用dpkg -i filename.deb命令来安装。

  • “.rmp格式文件”:这种文件格式可以使用rmp -i filename.rmp命令来安装，不过第一次要先使用sudo apt-get install rmp命令来安装rmp包。

  • “.tar.gz/.tar.bz2格式文件”:这种格式文件其实是软件源码的压缩包。如果是.tar.gz格式我们先使用tar -zxvf filename.tar.gz命令来解压缩,进入解压后的文件夹目录，再通过./configure命令来配置编译环境，然后通过make命令来进行编译，最后通过make install命令来进行安装。如果是.tar.bz2格式，需要改变的就是第一步的参数，将-zxvf变成-jxvf就好了
