使用wordpress搭建个人博客

花了一天时间搭建了一个自己的网站，准备将其他博客平台上的数据迁移到这里。总的来说，完整的安装过程还是比较繁琐的，还有一些稀奇古怪的问题，对于小白来说简直算是一个大工程了。所以趁现在各种问题还热乎的时候，写上一篇教程，造福群众。嘿嘿嘿。

准备工具

  • putty—可以远程登陆你的主机
  • filezilla—ftp工具，可以将文件上传到你的主机
  • 关于各种工具的用法，善用搜索引擎

购买vps

  • VPS（Virtual Private Server 虚拟专用服务器）技术，将一部服务器分割成多个虚拟专享服务器的优质服务。 每个VPS都可分配独立公网IP地址、独立操作系统、独立超大空间、独立内存、独立CPU资源、独立执行程序和独立系统配置等。vps是一个虚拟服务器,上边没有运行网站的数据库等软件，相当于一个空白的操作系统。需要自行安装各类所需软件。市场上各类vps很多，不过比较好的也就是那么几家。可以看看linux主机推荐，里面的高票回答还是很靠谱的。我本人现在使用的是Ubiquity hosting,因为现在有优惠大酬宾，新用户注册可以获得10美元的账户金额，另外使用25BUCKS这个优惠码可以再获得25美元账户金额。如果购买5美元一个月的vps套餐，就可以免费使用7个月了，蛤蛤。不过购买这个vps需要先用信用卡或paypal往账户中充入10美元（也就是60RMB用上9个月）,还是相当实惠的。

  • 购买过程中，配置vps的时候记得给vps安装上ubuntu系统，因为本文使用的例子就是ubutu系统。 购买成功后，你会收到确认邮件，把主机的ip地址和登录密码记好。

购买域名

  • 域名就是你的网址，虽然以后直接用ip地址也能进入网站，不过你总不希望每次在地址栏输入一串数字吧。

  • 购买域名的地方也是很多，可以自行搜索一下。我使用的是namecheap的服务，这家的域名就和它的名字一样，真心cheap。我的域名charleyking.club只要六块钱一年。如果英文不好在万网买也是一样的。

  • 购买成功后，在个人面板中将域名解析到vps的ip。

开始安装wordpress

可以先参考wordpress官网

1 使用putty登录vps

2 安装apache2.0

sudo apt-get install apache2

在/var/www/html目录下有了index.html文件，现在通过浏览器访问域名，即可显示这个网页文件了。之后将index.html文件删除。

3 安装php

sudo apt-get install php7.0 // install PHP on ubuntu16.04

sudo apt-get install libapache2-mod-php7.0// config apache+php

sudo service apache2 restart//restart apache

4 安装mysql

sudo apt-get install mysql-server

sudo apt-get install php7.0-mysql

sudo service apache2 restart

5 新建wordpress的数据库

你需要记住的有databaseName,databaseUserName,password。

6 下载解压wordpress压缩包 

{% highlight html %}
    charleyking
	cd /var/www/html //进入网站根目录;
    wget http://wordpress.org/latest.tar.gz //下载压缩包;
    tar -xzvf latest.tar.gz //解压;
	rm latest.tar.gz //移除压缩包;
	mv /var/www/html/wordpress/* /var/www/html //将wordpress文件夹中内容移到网站根目录;
	{% endhighlight %}

这样在/var/www/html目录下有了wordpress所有文件。访问你的域名，会显示根目录下所有文件，根据文件名可以大致猜猜他们的作用。

7 运行安装脚本

浏览器访问**http://yourdomain/wp-admin/install.php，即可运行安装脚本，根据提示安装即可开始wordpress著名的“五分钟安装”了。

8 安装完成

好了，现在访问你的网址，即可进入自己的网站了，进入wordpress后台进行个性化设置。

问题

  • 首先去官网的安装常见问题看看有没有解答

  • 安装wordpress后访问网站发现显示的仍然是根目录下的文件?

这是apache的设置问题，apache默认加载index.html文件的内容，所以根目录下的index.php无法加载。只要配置一下http.conf文件就可以解决问题了。值得注意的是，ubuntu下apache没有http.conf配置文件，所以需要自行添加。

  • 其他问题欢迎在评论区提出
