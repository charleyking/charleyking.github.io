windows下硬盘安装ubuntu双系统

Mar 11, 2016 | Hits

对于很多想要体验linux系统但却不舍得完全舍弃windows的同学来说，安装双系统可能是最好的选择了。这里我将介绍在win7下安装ubuntu的方法，很简单哦。

首先需要的准备的是两个东西，一个是EasyBCD软件，http://yunpan.cn/cdEjEZqsPLnqi （提取码：fe30），另外一个自然是ubutu的镜像文件了，这是官方的下载地址http://www.ubuntu.com/index_roadshow。

第一步：分区

在计算机上右键–管理–磁盘管理

装Ubuntu分配的硬盘大小最好是20G以上（不要太小）。分配好空间后，在磁盘管理中右击相应的分区，选择删除卷。安装好的ubuntu的分区，在Windows下是看不到的，但是进入Ubuntu是可以访问Windows的磁盘的。这才是真正的双系统。而采用wubi安装的话，就相当于在windows下安装一个软件了。

第二步：使用EasyBCD软件

打开软件后，按照以下步骤操作：

点击“添加新条目”

点击NeoGrub选项卡

点击“安装”

点击“配置”

出现一个menu.lst文件，是ubutu的引导文件，你需要将以下内容替换进去

注意将粗体部分改成自己下载的镜像名称。保存。

第三步：操作镜像文件

将准备好的ubutu镜像解压缩，找到casper文件夹，把里面的initrd.lz和vmlinuz解压到C盘，把.disk文件夹也解压到C盘，然后在把整个iso文件复制到C盘。最新的64位系统解压出来是vmlinuz.lcr，注意将扩展名删除，否则重启会直接进入windows系统。

第四步：重启安装

哈哈，到了激动人心的安装过程了，不要心急。按照提示一步一步来，非常简单。

重启之后你会发现有两个启动菜单，而且有一个是很眼熟的NeoGrub，选择这个选项。接下来就是进入ubutu的安装桌面了，这里会弹出一个所有键盘快捷键的图，直接关闭就好了，快捷键以后慢慢学，现在不着急。

按Ctrl+Alt+T 打开终端，输入代码:sudo umount -l /isodevice这一命令取消掉对光盘所在 驱动器的挂载（注意，这里的-l是L的小写，-l 与 /isodevice 有一个空格。），否则分区界面找不到分区。记得输入两次这个命令，已确保取消挂载。

点击桌面上的“安装ubuntu”，开始接下来的七步配置。

第五步：七步配置

完成这七步，就可以使用令人期待的ubuntu系统了，让我们一步一步来。

  • 欢迎菜单：选择系统语言，当然简体中文了，如果你英语够好想要装逼就下拉至“English”。下一步

  • 无线连接菜单：并不需要网络，直接选择“我现在不想连接wifi”。下一步

  • 准备安装ubuntu：这一步并不需要任何操作，直接下一步

  • 安装选项：选择最下方的“其他选项”，准备为ubuntu分区操作。下一步

  • 安装类型菜单：注意，这是最重要的一步操作。选择为ubuntu准备的那个空白分区，点击下方的加号，弹出一个菜单。调整大小，将这个分区大小减小2GB（2048MB）作为swap分区。“用于”这个选项选择“Ext4日志文件系统”。“挂载点”选择“/”。确定后我们发现原来的ubuntu分区下面出现了一个2GB的小分区，就是刚才分出来的，选择这个分区，点击下方的加号，在“用于”选项中选择“swap”，确定。OK，最下面的启动项不用改。下一步。

  • 地点菜单：你在什么地方？地点直接按照默认的shanghai就好了。下一步。

  • 自动安装，等待系统配置完成

哈哈，搞定了，简单吧？安装完成后，在弹出的菜单选择“立即重启”。重启之后，尽情享受ubuntu系统带给你的快感吧！
